# Getting Started with Posty

Welcome to your new Posty blog! This is a simple markdown-based blogging platform built with Node.js and Express.

## What is Posty?

Posty is a lightweight blog application that lets you:

- **Write posts in Markdown** - Use simple, readable formatting
- **Create posts easily** - No database needed, just markdown files
- **View your blog** - Beautiful, responsive design
- **Add new posts** - Simple form right in your browser

## How to Use

### Creating Posts

1. Click the **+ New Post** button on the homepage
2. Enter a title for your post
3. Write your content in Markdown
4. Click **Publish Post**

Your post will appear on the homepage and be accessible at its own URL.

### Markdown Tips

You can use all standard Markdown syntax:

- **Bold text**: `**bold**` or `__bold__`
- *Italic text*: `*italic*` or `_italic_`
- [Links](https://example.com): `[text](url)`
- `Code`: Use backticks for inline code
- Lists and more!

## File Structure

Posts are stored as markdown files in the `posts/` folder. Each post's URL slug is generated from the title automatically.

Happy blogging! 📝
